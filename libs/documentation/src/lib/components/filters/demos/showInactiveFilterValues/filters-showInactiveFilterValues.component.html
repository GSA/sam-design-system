Input isInactiveValueFieldShown has been added to enable a "Show Inactive Filter Values" checkbox being added to More Filters dialog. This input defaults to false. When enabled, an additional output showInactiveFiltersChange will become available for developers to connect to. This output will emit a boolean for the current state of the "Show Inactive Filter Values" checkbox.

The output emits when the user clicks "Update" and the value of "Show Inactive Filter Values" checkbox is different than when it opened.

In this example an option will be added/removed from the country select and a message will be logged when the showInactiveFiltersChange event emits, based on its value
