Setting Formly Autocomplete ID

This demo shows the multiple ways that ID can be set on autocomplete so that the label and autocomplete input are connected for accessibility purposes.

First Name shows how the ID can be dynamically set and still ensure the IDs match between label and input.

Middle Name shows a static ID being set and used by both label and autocomplete input.

Last Name shows the ID being auto-assigned by Formly and used by both label and autocomplete input.
