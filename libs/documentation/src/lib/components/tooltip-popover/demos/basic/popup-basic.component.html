When sdsPopover directive is applied to an element, a popover will appear when the element is clicked on. The title of this popover accepts either a string or element reference passed to sdsPopoverTitle. The content of this popover accepts either a string or element reference passed to the directive.

Please let us know what we could do better.

The position of the popover can be manipulated by the postion input. By default, the popover will appear at the top of the element.

Valid position values include top, right, bottom, left

The sdsPopoverTitle input is not required. In the case where no value is provided to the title input a popover with no divider will be rendered.
