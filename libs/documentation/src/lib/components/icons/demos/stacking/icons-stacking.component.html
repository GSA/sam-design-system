To stack two icons, apply the sds-stack class the parent HTML element of the icons you want to stack. Then add the sds-stack-icon class to the icons you want stacked.

Classes starting with sds- and followed by a i-uswds size multiplier ie: sds-sm or sds-2x can be applied to the parent HTML element to scale the overall size of the stacked icons. For further refinement of one of the icons, the size input of i-uswds can be used.
