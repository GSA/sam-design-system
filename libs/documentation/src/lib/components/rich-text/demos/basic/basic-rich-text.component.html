Inputs

maxHeight:number (Optional) - Minimum height in rems that the editor can be. If text entered by the user expands beyond the bottom of the editor, the editor will expand to display all text. This value is expected to be an positive integer or zero. The minimum value is 0 which will provide enough room for a single paragraph line to appear. The maximum value is X which is large enough to allow for 10 lines of h1 text.
