In order to stack two or more icons, StackedIconModule must be imported into the module where you wish to use a stacked icon.

import { NgxBootstrapIconsModule, square } from 'ngx-bootstrap-icons';
import { add, IconModule, StackedIconModule } from '@gsa-sam/ngx-uswds-icons';

@NgModule({
 imports: [
  CommonModule,
  IconModule,
  StackedIconModule,
  NgxBootstrapIconsModule.pick({add, square})
 ],
 declarations: [YourComponent],
 exports: [YourComponent]
})

After importing the module you can update your template to include <usa-stacked-icon>. The only input for this component is size which can be used to scale the stacked icon using the same values as those used to modify the size of individual icons.

The icons you want to appear stacked should then be inserted as children of <usa-stacked-icon>. In order for the stacking css to work, make sure that the class usa-stack-icon is applied to the icons you are stacking.

<usa-stacked-icon [size]="'sm'">
 <usa-icon class="usa-stack-icon" [size]="'2x'" [icon]="'square-fill'"></usa-icon>
 <usa-icon class="usa-stack-icon text-base-lightest" [size]="'lg'" [icon]="'hand-thumbs-down'"></usa-icon>
</usa-stacked-icon>
