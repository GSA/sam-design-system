The base input requirements for a selection panel component is a [model] of type SelectionPanelModel. This model extends off of SideNavigationModel and contains an additional property called selectionMode, which can either be 'SELECTION' or 'NAVIGATION' ('NAVIGATION' is default mode). On 'SELECTION' mode, the children of navigation links will not be shown while on 'NAVIGATION' mode they will. By default, 'NAVIGATION' mode allows internal route navigation on selection, while 'SELECTION' mode does not

The input [navigateOnClick] is of type boolean and can be used to toggle on/off route navigation when a panel item is selected.

An output event (panelSelected) is fired anytime a navigation item is selected with the selected NavigationLink
